  [Summary]
    Unity Meshes combiner.
--------------------------------------------------------------------------
  [Demand]
    In Unity scene, a large number of static models(example stones) spend
    lots of computer resource and hard to manage them in Unity Hierarchy.
    therefore, we hope that we can combine them to a model(Mesh).
--------------------------------------------------------------------------
  [Environment]
    Unity 5.0 or above.
    .Net Framework 3.5 or above.
--------------------------------------------------------------------------
  [Achieve]
    MeshCombiner：Draw the extend editor window and combine Meshes.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Mesh Combiner" in Unity editor menu bar and
    click it or press key combination Alt+M to open the editor window.

    Create an empty gameobject as "Meshes Root", drag the target meshes
    gameobjects to under the "Meshes Root".

    Create an empty gameobject as "Mesh Save".

    Click the "Combine" button to open the save new mesh dialog.

    Select a path and input a file name to the dialog and save.
